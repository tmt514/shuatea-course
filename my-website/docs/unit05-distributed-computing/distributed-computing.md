# 分散式計算的思考方式

:::tip 核心概念
有別於平行計算時的記憶體共享模式，在分散式系統內部節點與節點之間的溝通都是透過網路進行的。也就是說，如果能將存取的資料與相依性切得越開，演算法就會變得更有效率。
:::
