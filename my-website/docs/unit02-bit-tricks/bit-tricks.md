# 位元運算 Bitwise Operators

:::tip 核心觀念
1. 利用單位時間能對位元組進行操作的特性，加速某些計算。
2. 「狀態壓縮」：利用位元組來表達資料集合的概念。
:::

* 2.2 枚舉所有子集合
