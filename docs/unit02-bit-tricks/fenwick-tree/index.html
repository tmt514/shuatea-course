Fenwick 樹

Fenwick 樹是一種資料結構，它藉由位元計算的技巧，能夠幫助我們快速回答一些關於「前綴」的加總問題。

首先，讓我們來定義一下這個資料結構能做到的事情：

單點改值與前綴加總問題

對於一個固定長度的正整數 nnn，我們欲對一個長度為 nnn 的序列 x1,x2,…,xnx_1, x_2, \ldots, x_nx1​,x2​,…,xn​ 進行以下操作：

  • add(i, value): 將 xix_ixi​ 的值改變為 xi+valuex_i + {\tt{value}}xi​+value。
  • prefix_sum(r): 回答 x1+x2+⋯+xrx_1+x_2+\cdots + x_rx1​+x2​+⋯+xr​ 的值。

Fenwick Tree 的定義還滿單純的，它利用一個陣列就能夠在 O(log⁡n)O(\log n)O(logn) 的時間內完成上面兩項操作。
